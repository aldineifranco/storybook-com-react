# Global Style

Estilo base do CSS das aplicações:

```jsx
const App = () => (
  <>
    <GlobalStyle />
    <OtherComponents />
  </>
);
```

## Tipografia

### Títulos

Os títulos foram configurados de `<h1>` até `<h6>`:

<Preview>
  <Story name="Headings">
    <div>
      <h1>Título 1</h1>
      <h2>Título 2</h2>
      <h3>Título 3</h3>
      <h4>Título 4</h4>
      <h5>Título 5</h5>
      <h6>Título 6</h6>
    </div>
  </Story>
</Preview>

### Parágrafos
